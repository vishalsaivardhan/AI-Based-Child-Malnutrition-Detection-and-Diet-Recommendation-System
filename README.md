# AI-Based-Child-Malnutrition-Detection-and-Diet-Recommendation-System
A machine learning system that predicts malnutrition levels in children using health parameters and provides personalized diet recommendations based on nutritional needs.

This project was developed as a **Community Service Project (CSP)** to assist healthcare workers in identifying malnutrition levels in children and providing immediate dietary recommendations.

## 🚀 Features
- **Predictive Engine:** Uses a Random Forest Classifier (91.4% accuracy).
- **Personalized Diet:** Recommends high-protein/calorie Indian dishes from a processed dataset.
- **Web Interface:** Built with Streamlit for easy use in the field.

## 🛠️ Tech Stack
- Python (Pandas, Scikit-Learn)
- Streamlit (Web UI)
- Kaggle Datasets (Malnutrition & Indian Food Nutrition)

## 📦 How to Run
1. Clone the repo: `git clone <your-repo-link>`
2. Install requirements: `pip install pandas scikit-learn streamlit`
3. Run the app: `streamlit run app.py`
